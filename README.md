# Docker-stury-for-rails

## Ruby version
```bash
2.6.1
```

## Rails version
```bash
5.2.2
```

## Database creation
```bash
$ bundle exec rails db:create
```

## Database initialization
```bash
$ bundle exec rails db:migrate
```

## How to run the test suite
```bash
$ bundle exec rails test
```

## Execute rails server
```bash
$ bundle exec rails s
```
